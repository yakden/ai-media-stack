# Iris — Video Management System

> **Iris is now a separate, private commercial product.**
> A public repository (community / commercial edition) is planned — this page
> will link to it once it is available.

**Iris** is a self-hosted, single-GPU VMS with cross-camera, **face-anchored**
person & object re-identification. It began here, inside this platform, and has
since grown into a standalone product: low-latency live monitoring, person-
triggered recording, automatic clip subtitles + translation, multi-protocol
sources (RTSP / RTMP / HLS / SRT / MPEG-TS / DASH / web links) and neural
voice-activity detection.

Its source and documentation now live outside this open-source monorepo. It
still runs on the same shared NVIDIA T4 alongside the rest of this platform.
